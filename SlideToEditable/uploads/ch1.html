What is Machine Learning?

Что такое машинное обучение?

Definition: Machine learning is both the application and the science of algorithms that can make some sense out of data.

Определение: Машинное обучение — это одновременно приложение и наука об алгоритмах, которые могут осмысливать данные.

Purpose: It allows data to be used to create knowledge and predict future events.

Цель: Оно позволяет использовать данные для создания знаний и предсказания будущих событий.

💡 Machine learning helps computers automate, anticipate and evolve, but still needs human beings to set parameters and train the machine.

💡 Машинное обучение помогает компьютерам автоматизировать, предвидеть и эволюционировать, но всё ещё нуждается в людях для настройки параметров и обучения машины.

The Role of Algorithms

Роль алгоритмов

Algorithm: A method of analyzing data in a way that makes it possible for machines to analyze and process data.

Алгоритм: Метод анализа данных, который позволяет машинам анализировать и обрабатывать информацию.

Capabilities: Computers can understand different concepts, make choices, and predict possibilities for the future.

Возможности: Компьютеры могут понимать концепции, делать выбор и предсказывать возможности.

⚠️ Important: Without algorithms, machines cannot learn. They enable computers to turn data into knowledge for data-driven decisions.

⚠️ Важно: Без алгоритмов машины не могут учиться. Они позволяют компьютерам превращать данные в знания для принятия решений.

Three Types of Machine Learning

Три типа машинного обучения

1. Supervised Learning

1. Обучение с учителем

  • Requires labeled training data
  • Требует размеченных обучающих данных

2. Unsupervised Learning

2. Обучение без учителя

  • Uses unlabeled data with unknown structure
  • Использует неразмеченные данные

3. Reinforcement Learning

3. Обучение с подкреплением

  • Agent improves through environment interaction
  • Агент улучшается через взаимодействие со средой

Supervised Learning

Обучение с учителем

How it works: Requires a detailed input of related data over a period of time. Once all information is available, it classifies new data.

Как работает: Требует подробного ввода связанных данных за период времени. После получения всей информации классифицирует новые данные.

Process: The computer performs calculations, comparisons, and analysis before making a decision.

Процесс: Компьютер выполняет вычисления, сравнения и анализ перед принятием решения.

👤 Human Input: Requires human input to ensure the computer has the right data to process. We already know the answer we want when the model is being trained.

👤 Участие человека: Требует участия человека для обеспечения правильности данных. Мы уже знаем желаемый ответ при обучении модели.

Unsupervised Learning

Обучение без учителя

Challenge: The computer does not have all the information to make a decision. Some factors may be missing.

Сложность: У компьютера нет всей информации для принятия решения. Некоторые факторы могут отсутствовать.

Solution: Must be equipped with tools to find missing elements without human help. More complex than supervised algorithms.

Решение: Должно быть оснащено инструментами для поиска недостающих элементов без помощи человека. Сложнее алгоритмов с учителем.

Techniques: Clustering and Dimensionality Reduction

Техники: Кластеризация и уменьшение размерности

🔍 Clustering: Organizes data into subgroups or clusters based on similarity. Often termed as "unsupervised classification".

🔍 Кластеризация: Организует данные в подгруппы или кластеры на основе сходства. Часто называется «классификацией без учителя».

Reinforcement Learning

Обучение с подкреплением

Goal: Develop an agent or system that can improve its performance based on interaction with its environment.

Цель: Разработать агента или систему, которая может улучшить свою производительность на основе взаимодействия со средой.

Reward System: Uses a reward-signal (positive or negative) defined by achieving a specific goal.

Система вознаграждения: Использует сигнал вознаграждения (положительный или отрицательный), определяемый достижением цели.

🎮 How it differs: With supervised learning, we know the answer. With Reinforcement Learning, we provide a reward measured by specific actions performed by the agent.

🎮 Отличие: При обучении с учителем мы знаем ответ. При обучении с подкреплением мы предоставляем вознаграждение за конкретные действия агента.

Data Preprocessing & Training

Предварительная обработка и обучение

Preprocessing: Getting data into shape. Raw data is rarely in the right format for learning algorithms.

Предобработка: Приведение данных в форму. Сырые данные редко в правильном формате для алгоритмов.

Feature Scaling: Features need identical scales. Achieved by transforming to a range or using standard distribution.

Масштабирование: Признаки нуждаются в одинаковых масштабах. Достигается преобразованием к диапазону или стандартному распределению.

Dataset Split: Divide into Training and Test sets. Training trains the model, test evaluates it.

Разделение: Разделите на обучающий и тестовый наборы. Обучающий обучает модель, тестовый оценивает.

⚙️ Hyperparameter Optimization: Fine-tune model performance. Hyperparameters are dials we can turn to improve the model (cannot be learned from dataset).

⚙️ Оптимизация гиперпараметров: Настройка производительности модели. Гиперпараметры — это регуляторы для улучшения модели (не обучаются из данных).

Dimensionality Reduction

Уменьшение размерности

Purpose: Remove noise from data that can cause degradation in predictive performance.

Цель: Удалить шум из данных, который может ухудшить прогнозную производительность.

Benefits: Compress data to fit smaller subspace while keeping relevant information. Reduces storage space and makes algorithms faster.

Преимущества: Сжатие данных в меньшее подпространство с сохранением информации. Уменьшает место и ускоряет алгоритмы.

📊 Visualization: Project high-dimensional feature set onto 1, 2, or 3D spaces. Allows viewing through 2D or 3D histograms or scatter plots.

📊 Визуализация: Проекция многомерного набора признаков на 1, 2 или 3D пространства. Позволяет просматривать через 2D или 3D гистограммы.

Training and Choosing Models

Обучение и выбор моделей

Model Selection: Several algorithms are compared to select the best model. Common metric: classification accuracy.

Выбор модели: Сравниваются несколько алгоритмов для выбора лучшей модели. Общая метрика: точность классификации.

Cross-Validation: Dataset split into training and validation subsets to estimate generalization performance.

Кросс-валидация: Набор данных разделяется на обучающую и проверочную подмножества для оценки обобщающей способности.

⚠️ Important Warning: Parameters for procedures (dimensionality reduction, feature scaling) must be obtained ONLY from training dataset, then applied to test dataset. Otherwise, test data may produce overly-optimistic performance measurement!

⚠️ Важное предупреждение: Параметры для процедур (уменьшение размерности, масштабирование) должны получаться ТОЛЬКО из обучающего набора, затем применяться к тестовому. Иначе тестовые данные могут дать излишне оптимистичную оценку!
