# Справочный Робот - Многофункциональный интеллектуальный помощник

Интеллектуальный помощник, предоставляющий функции прогноза погоды, конвертации валют, перевода слов, калькулятора и конвертера единиц, поддерживающий голосовое воспроизведение и голосовые команды, все результаты сохраняются в базу данных.

## Функциональные возможности

### 🌤️ Прогноз погоды
- Запрос информации о погоде в городах по всему миру
- Отображение подробной информации: температура, влажность, скорость ветра и т.д.
- Поддержка использования текущего местоположения

### 💱 Конвертация валют
- Поддержка конвертации в реальном времени для различных валют
- Включает доллар США, китайский юань, евро, японскую иену, британский фунт, российский рубль и другие
- Отображение реального курса и обратного курса

### 🌐 Функции перевода
- Поддержка взаимного перевода между несколькими языками
- Включает китайский, английский, японский, корейский, французский, немецкий, испанский, русский и другие языки
- Автоматическое определение языка

### 🧮 Калькулятор
- Поддержка основных математических операций
- Поддержка расширенных операций: тригонометрические функции, логарифмические функции и т.д.
- Поддержка констант (π, e) и факториальных операций

### 📏 Конвертация единиц
- Конвертация единиц длины, веса, температуры, объема, скорости, площади, времени, хранения данных
- Поддержка международных единиц и традиционных китайских единиц
- Расчет конвертации в реальном времени

### 🎤 Голосовые функции
- Преобразование текста в речь (поддержка нескольких языков)
- Преобразование речи в текст (распознавание речи)
- Ввод голосовых команд
- Загрузка и распознавание аудиофайлов

### 📊 История операций
- Все операции автоматически сохраняются в базу данных
- Фильтрация истории по типам операций
- Функция очистки истории

## API интерфейсы

### Прогноз погоды
- **POST** `/api/weather`
- Параметры: `{"city": "Название города"}`

### Конвертация валют
- **POST** `/api/currency`
- Параметры: `{"from": "Исходная валюта", "to": "Целевая валюта", "amount": Сумма}`

### Перевод
- **POST** `/api/translate`
- Параметры: `{"text": "Текст", "source": "Исходный язык", "target": "Целевой язык"}`

### Калькулятор
- **POST** `/api/calculate`
- Параметры: `{"expression": "Математическое выражение"}`

### Конвертация единиц
- **POST** `/api/convert-units`
- Параметры: `{"value": Числовое значение, "from_unit": "Исходная единица", "to_unit": "Целевая единица", "category": "Категория"}`

### Преобразование текста в речь
- **POST** `/api/text-to-speech`
- Параметры: `{"text": "Текст", "language": "Код языка"}`

### Преобразование речи в текст
- **POST** `/api/speech-to-text`
- Параметры: Аудиофайл (multipart/form-data)

### История операций
- **GET** `/api/history`
- Опциональные параметры: `?operation_type=тип`

### Вопросы и ответы DeepSeek
- **POST** `/api/deepseek-ask`
- Параметры: `{"question": "Вопрос", "context": "Контекст (опционально)"}`

## Конфигурация

### Переменные окружения
Создайте файл `.env` (опционально):
```
SECRET_KEY=your-secret-key
TRANSLATION_APP_ID=your-baidu-app-id
TRANSLATION_API_KEY=your-baidu-api-key
DEEPSEEK_API_KEY=your-deepseek-api-key
```

### Конфигурация DeepSeek API
1. Зарегистрируйте аккаунт DeepSeek и получите API-ключ
2. Установите переменную окружения: `export DEEPSEEK_API_KEY='your-api-key'`
3. Или добавьте в файл `.env`: `DEEPSEEK_API_KEY=your-api-key`
4. Если API-ключ не установлен, система автоматически использует имитационные данные или традиционные API

### База данных
- Используется база данных SQLite, автоматически создается файл `reference_robot.db`
- Содержит таблицу `history` для хранения всех записей операций

## Инструкции по разработке

### Добавление новых функций
1. Создайте новый модуль в каталоге `api/` или `utils/`
2. Добавьте соответствующий маршрут API в `app.py`
3. Добавьте соответствующую JavaScript функцию в `static/js/main.js`
4. Добавьте соответствующий интерфейс в `templates/index.html`

## Важные замечания

1. **Голосовые функции**: Требуется установка дополнительных голосовых библиотек, если не установлены, голосовые функции будут использовать имитационные данные
2. **Внешние API**: Функции перевода, погоды и курсов валют используют имитационные данные, для реальных данных необходимо настроить соответствующие API-ключи
3. **DeepSeek API**: AI-функции требуют настройки переменной окружения DEEPSEEK_API_KEY, в противном случае используются имитационные данные или традиционные API
4. **База данных**: Все результаты операций автоматически сохраняются в базу данных SQLite
5. **Безопасность**: В производственной среде установите надежные пароли и HTTPS
